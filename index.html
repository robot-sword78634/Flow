Flow

Friendly Language of Wonder

What is Flow?

Flow is a small, friendly, educational programming language inspired by classic 1970s and early 1980s computing. It is designed to make programming approachable, readable, and fun.

Flow leans heavily into the simplicity of early programming languages such as BASIC, but utilizes a model based on visual flow chart logic rather than line numbers. The language is equipted with a robust set of mathematical functions that should suit most users needs and is capable of handling simple string variables. The idea behind the creation of Flow was to create an interpreted language that was low barrier for individuals who were new to programming and would inspire them to push the language's limitations to build new and interesting things. In short, to create a "friendly" language that would inspire wonder and interest in learning about more complicated concepts. Essentially the same experience I had when first sitting in front of a computer in 1984 and realizing that I now had a set of tools that I could use to create anything I wanted.

Flow encourages learning through experimentation, creativity, and play rather than complexity or performance.

Philosophy

  • Simple syntax that is easy to read and understand
  • Intuituve design for people new to programming
  • Designed for games, simulations, and experiments
  • Intentional limitations that encourage creativity
  • Learning by doing

What Can You Make With Flow?

  • Text adventure games
  • Classic computer simulations (like Lunar Lander)
  • Math tutors and learning tools
  • Interactive demos and showcases
  • Terminal-based games and experiments

Downloading & Compiling Flow

Clone the Flow repository from GitHub: 

git clone https://github.com/robot-sword78634/flow.git
cd flow

Compile the interpreter (example using g++): 

g++ -std=c++11 flow.cpp -o flow

(Adjust filenames and compiler options as needed for your system.)

Hello World in Flow

Create a simple text file called hello.flow:

print "Hello, world!"

Run it using the Flow interpreter:

./flow hello.flow

Congratulations — you’ve written your first Flow program!

The History of Flow

    The development of Flow came from some limited success I had creating a simple Lisp interpreter and a fantastic failure I encountered when trying to expand my coding skills into creating a modern version of BASIC. It occurred to me that a much easier project would be to create my own version of the best things I liked about early languages with a smattering of some of the things that make modern languages as accessible as they are. As I began mapping out the language it occured to me that a language that read more like pseudocode than traditinoal code might make for a useful teaching tool when used to introduce people to computer programming. Most of the syntax used in Flow are stripped down versions of what people utilize in modern languages, with the idea that the program should "flow" like the logic paths in a flow chart. Flow is written in C++ and implemented as an interpreted language, which makes it portable and much easier to work with without having to get too "close to the metal." Some future ideas for implentation include:

  • Expanding ASCII output abilities
  • Adding color
  • Implementing arrays/lists
  • Building the ability to create user defined functions
